    athackst/dockerfiles

    These are multi-stage docker images I use for developing with VSCode.

    See how I develop with vscode and docker.

    Instructions on how to duplicate my IDE:

    • vscode, docker, and ros2
    • vscode, docker, and github pages

    Quick start

    Grab the docker image from docker hub

    • athackst/ros
    • athackst/ros2
    • athackst/gazebo
    • atahckst/igntion
    • athackst/github

    Set up a vscode workspace

    Build the dockerfiles

    Create the docker images

    ./build.py all

    To see help information on the update script

    ./build.py --help
